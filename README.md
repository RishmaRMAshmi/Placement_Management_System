# Placement Management System

A Flask-based web application developed to streamline campus recruitment processes for students, companies, and administrators.

## Features
- Role-based login for Admin, Student, and Company
- Company registration and approval system
- Job drive creation and management
- Student application tracking
- Resume upload functionality
- Application status updates (Selected, Rejected, Shortlisted)
- Admin dashboard for managing users and drives

## Tech Stack
- Python
- Flask
- SQLite
- SQLAlchemy
- HTML
- CSS
- Bootstrap

## Installation

```bash
pip install -r requirements.txt
python app.py
